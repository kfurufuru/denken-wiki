    頻出数値一覧¶

    電験3種 法規で問われる数値を一覧化。✅ マークは法令原文との照合確認済み。

    接地抵抗値¶

    接地種別 接地抵抗値 適用 備考
    A種接地 ✅ 10 [Ω] 以下 高圧・特別高圧機器の外箱 —
    B種接地 ✅ 150/Ig [Ω] 以下 高圧・低圧混触防止 Ig = 1線地絡電流 [A]
    C種接地 ✅ 10 [Ω] 以下 300Vを超える低圧機器の外箱 0.5秒以内に遮断: 500Ω以下
    D種接地 ✅ 100 [Ω] 以下 300V以下の低圧機器の外箱 0.5秒以内に遮断: 500Ω以下

    絶縁抵抗値（低圧電路）¶

    省令第58条に基づく3区分（電技解釈 第14条）

    電路の区分（省令第58条） 絶縁抵抗値
    最大供給電圧300V以下・対地電圧150V以下（接地式電路） ✅ 0.1 [MΩ] 以上
    最大供給電圧300V以下・その他（非接地式・対地電圧150V超300V以下） ✅ 0.2 [MΩ] 以上
    最大供給電圧300V超 ✅ 0.4 [MΩ] 以上

    接地線の太さ¶

    接地種別 接地線の最小太さ 備考
    A種接地 ✅ 2.6mm（軟銅線） —
    B種接地 ✅ 4mm以上（Igによる） 1線地絡電流による
    C種接地 ✅ 1.6mm（軟銅線） —
    D種接地 ✅ 1.6mm（軟銅線） —

    架空電線の高さ（地表上）¶

    場所・条件 低圧 高圧
    道路横断 ✅ 6m 以上 ✅ 6m 以上
    鉄道横断 ✅ 5.5m 以上 ✅ 5.5m 以上
    横断歩道橋の上 ✅ 3.5m 以上 ✅ 3.5m 以上
    道路以外（一般） ✅ 5m 以上 ✅ 5m 以上
    道路以外（低圧・技術上やむを得ない場合） ✅ 4m 以上 —

    ✅ 上記は代表値。ケーブル使用時・絶縁電線使用時で緩和条件あり。

    離隔距離¶

    架空電線と建造物¶

    電線の種類 離隔距離
    低圧（絶縁電線） ✅ 1m 以上（上方）/ 0.6m以上（側方・下方）
    高圧（絶縁電線） ✅ 1.2m 以上
    高圧（ケーブル） ✅ 0.6m 以上

    架空電線相互間¶

    組み合わせ 離隔距離
    低圧 と 低圧 ✅ 0.3m 以上
    低圧 と 高圧 ✅ 0.6m 以上
    高圧 と 高圧 ✅ 0.8m 以上

    条件付き緩和あり（ケーブル使用時等）。照合時は個別条文を確認すること。

    遮断器の動作時間¶

    過電流遮断器（低圧）¶

    定格電流 1.25倍 2倍
    30A以下 ✅ 60分以内 ✅ 2分以内
    30A超〜50A以下 ✅ 60分以内 ✅ 4分以内
    50A超〜100A以下 ✅ 120分以内 ✅ 6分以内
    100A超〜225A以下 ✅ 120分以内 ✅ 8分以内

    漏電遮断器（地絡遮断装置）¶

    項目 数値
    感度電流 ✅ 定格感度電流（15mA, 30mA等）
    動作時間 ✅ 0.1秒以内（高速型）

    対地電圧の制限¶

    場所 制限 条件
    住宅の屋内電路 ✅ 対地電圧 150V 以下 原則
    住宅以外 ✅ 対地電圧 150V 以下（原則） 定格消費電力2kW以上の機器等は300V以下

    簡易接触防護措置を講じた場合等の緩和条件あり。

    地中電線路の埋設深さ¶

    施設方法 車両通行がある場所 その他
    直接埋設式 ✅ 1.2m 以上 ✅ 0.6m 以上
    管路式 ✅ 1.2m 以上 ✅ 0.3m 以上

    支持物の安全率¶

    支持物の種類 安全率
    木柱 ✅ 2.0 以上
    鉄柱 ✅ 2.0 以上
    鉄筋コンクリート柱 ✅ 2.0 以上
    鉄塔 ✅ 2.0 以上

    その他の重要数値¶

    項目 数値 備考
    電圧区分（低圧） ✅ 直流750V以下 / 交流600V以下 —
    電圧区分（高圧） ✅ 直流750V超〜7,000V以下 / 交流600V超〜7,000V以下 —
    電圧区分（特別高圧） ✅ 7,000V超 —

    工事計画届出の期限 ✅ 工事開始30日前 電気事業法第48条

    事故報告（速報） ✅ 24時間以内 電気関係報告規則
    事故報告（詳報） ✅ 30日以内 電気関係報告規則

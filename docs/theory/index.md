## 理論を攻略する3つの視点

1. **距離依存を意識する** — 点電荷・直線電流・面電荷でそれぞれ違う
2. **因果の向きを固定する** — 何が原因で何が結果かを先に言えるか確認
3. **ベクトルの向きは右手則一択** — 計算前に向きを決めてから大きさを求める

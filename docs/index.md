# 電気設備技術Wiki

> 電験3種 法規 — 条文×過去問クロスリファレンス

## このWikiの目的

条文を**暗記する**のではなく、**なぜその法令が必要か**を物理・現場・法規の3層で理解する。

```
🟥 法律（電気事業法）     ← なぜ規制するのか
🟧 政令（施行令）         ← 規制の範囲
🟨 省令（技術基準）       ← 何を守るか
🟩 告示・解釈（電技解釈）  ← どうやって守るか
🟦 規格/ガイドライン（JIS等）← 具体的な数値・方法
```

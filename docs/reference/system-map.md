## 日常の学習フロー

```
📖 勉強する
 │
 ├─→ denken-wiki で条文・テーマを確認（Layer 1）
 │
 ├─→ 過去問を解く
 │    └─→ denken3-study に結果を自動記録（Layer 2）
 │
 ├─→ 間違えた！
 │    ├─→ .secretary/e-log に認知バグを記録（Layer 3）
 │    └─→ 「フェインマン [テーマ]」で理解度を確認
 │
 └─→ 理解が深まった
      ├─→ denken-wiki に解説を追記・⚠️→✅に更新
      └─→ 「理解度更新 [ファイル] [スコア]」で知識代謝を更新
```

## どこに何がある？

| やりたいこと | 行く場所 | URL/コマンド |
|-------------|---------|-------------|
| B種接地の緩和条件を確認 | denken-wiki | テーマ別 → 接地工事 |
| 今年の法規の過去問を確認 | denken-wiki | 過去問 → 年度別 |
| 理論の達成率を確認 | denken3-study | ダッシュボード |
| テスト結果を記録 | ChatGPT | 「記録して」→ 自動連携 |
| 間違えた理由を分析 | .secretary | 「誤答記録」コマンド |
| 本当に理解しているか確認 | .secretary | 「フェインマン 接地工事」 |
| 復習すべきテーマを確認 | .secretary | 「知識チェック」 |
| e-Govで条文原文を確認 | denken-wiki | リファレンス → 参考文献 |
| システムの健全性を確認 | .secretary | 「ひろゆき巡回」 |

## 管理体制

| 役割 | 担当 | 責任範囲 |
|------|------|---------|

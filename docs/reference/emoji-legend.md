# 絵文字マーカー凡例

> このWikiで使用する絵文字マーカーの一覧と意味。

---

## 内容マーカー

| 絵文字 | 意味 | 使用場面 |
|--------|------|---------|
| 😱 | 事故・危険 | 感電事故・重大インシデントの記述 |
| 💀 | 死亡事故・致命的リスク | 感電死・死亡に至る条件の記述 |
| 🔥 | 火災 | 短絡・過負荷による火災リスクの記述 |
| 💥 | 爆発 | 可燃性ガス・粉じんによる爆発リスクの記述 |
| 🧠 | 思考プロセス | 記憶すべき考え方・理屈の説明 |
| 🔍 | 注目ポイント | 特に注意して読むべき箇所 |
| 🕳️ | 落とし穴 | 試験で引っかかりやすい罠・紛らわしい選択肢 |
| 🎯 | 重要ポイント | 試験に直結する最重要事項 |
| 🔗 | 関連条文リンク | 他の条文・テーマへの参照 |
| 👑 | 王道の考え方 | 法令の根底にある基本原則・設計思想 |
| ⚡ | 5秒で思い出す | 速攻暗記用のフレーズ・語呂合わせ |

---

## ステータスマーカー

| 絵文字 | 意味 | 使用場面 |
|--------|------|---------|
| ✅ | 確認済み | e-Gov法令検索 + 参考書 + 本Wiki の3点が一致した数値・条文 |
| ⚠️ | 未確認 | AIの知識に基づく参考値。e-Gov法令検索との突合が未完了 |
| 📄 | 作成済み | ページが作成完了し内容がある状態 |
| 📝 | 作成予定 | ページ未作成・今後対応予定 |

---

## 出題頻度マーカー

| マーカー | 意味 | 出題頻度の目安 |
|---------|------|---------------|
| ★★★★★ | 超頻出 | 毎年〜隔年で出題 |
| ★★★★☆ | 頻出 | 2〜3年に1回出題 |
| ★★★☆☆ | 標準 | 3〜5年に1回出題 |
| ★★☆☆☆ | 低頻度 | 過去11年で1〜2回 |
| ★☆☆☆☆ | まれ | 過去11年でほぼ出題なし |

## 法体系カラーマーカー

|---------|-----------|--------|
| 🟥 | 法律 | 電気事業法 |
| 🟧 | 政令 | 電気事業法施行令 |

| 🟦 | 規格 | JIS・IEC |

---

## 使い方のルール

1. **1つの段落に絵文字は2つまで** -- 多用すると視認性が下がる
2. **ステータスマーカーは必ずつける** -- ✅ か ⚠️ で数値の信頼性を明示
3. **法体系カラーは条文引用時に使う** -- どのレベルの法令かを一目で判別

# 電気事業法 第55条 — 定期安全管理検査

> 特定電気工作物（発電用ボイラー・タービン等）の設置者は、定期的に事業者検査を実施し技術基準への適合を確認しなければならない。実施体制は主務大臣の審査を受ける

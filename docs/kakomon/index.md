## 出題パターンの傾向（H25〜R07 分析済み）

- **A問題**（問1〜問10, 各5点 = 50〜55点）: 穴埋め・論説が中心。条文の正確な知識を問う
- **B問題**（問11〜問13, 各10〜14点 = 26〜42点）: 計算問題が中心。配点が大きく合否を左右

### 典型的な出題配置

| 問番号 | 出題パターン | 該当テーマ（頻度順） |
|--------|------------|-------------------|
| 問1〜2 | 電気事業法・関連法規 | 電気事業法（毎年）/ 事故報告・工事士法 |
| 問3〜5 | 省令・解釈の基本条文 | 接地・絶縁・保護装置・風車・地中 |
| 問6〜9 | 解釈の各論（施設規定） | 架空・配線・分散型電源・EV給電設備・特殊場所 |
| 問10 | 電気施設管理（知識） | 保護協調・受電設備・変流器・開閉器 |
| 問11〜13 | 電気施設管理（計算） | 絶縁耐力・接地抵抗・力率改善・水力・高調波 |

### 近年の傾向変化（R04〜R07）

- **分散型電源・系統連系**: R04以降ほぼ毎年出題。解釈§227（高圧連系保護装置）は最重要条文
- **電気事業法**: R04から年2回化で出題幅が拡大。需給制限（§34）・事故報告（報告規則§3）が頻出
- **EV・再エネ新技術**: R07下期に「電気自動車から電気を供給する設備」（解釈§220の2）が初出題。今後増加傾向
- **計算問題**: 絶縁耐力試験（解釈§15）・接地抵抗計算（解釈§17）が定番化。水力発電・高調波計算も増加
- **国際規格対応**: R07下期に省令§4の2（国際規格の取り入れ）が出題。改正に注意

### R08以降の出題予測

- 省エネ法・電気事業法改正（再エネ拡大関連）からの新出条文に注意
- EV給電・蓄電池関連の施設規定（解釈§220系）は継続出題が見込まれる
- 風車・太陽電池の保護装置は計算・論説ともに頻出維持

## 使い方

### 1. 過去問を解いたら記録する

各年度の列に出題があれば `○` を記入。問題番号も添える。

例: `○(B7)` → その年のB問題 問7で出題

### 2. テーマページから逆引きする

各テーマページの末尾に「過去問出題履歴」セクションを設け、具体的な問題内容を記録する。

## 出題パターンの傾向（分析予定）

- **A問題**（各2点 x 13問 = 26点）: 穴埋め・正誤が中心。制度・定義・数値を問う
- **B問題**（各6〜7点 x 3問 = 24点）: 計算問題が中心。電気施設管理（需要率・負荷率・力率改善）が定番

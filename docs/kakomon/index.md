## 出題パターンの傾向（H25〜R03 分析済み）

- **A問題**（問1〜問10, 各5点 = 50〜55点）: 穴埋め・論説が中心。条文の正確な知識を問う
- **B問題**（問11〜問13, 各10〜14点 = 26〜42点）: 計算問題が中心。配点が大きく合否を左右

### 典型的な出題配置

| 問番号 | 出題パターン | 該当テーマ（頻度順） |
|--------|------------|-------------------|
| 問1〜2 | 電気事業法・関連法規 | 電気事業法（毎年） |
| 問3〜5 | 省令・解釈の基本条文 | 接地・絶縁・保護装置・地中 |
| 問6〜9 | 解釈の各論（施設規定） | 架空・配線・分散型電源・特殊場所 |
| 問10 | 電気施設管理（知識） | 保護協調・受電設備・需給管理 |
| 問11〜13 | 電気施設管理（計算） | 風圧荷重・力率改善・需要率・%Z |

### 近年の傾向変化

- **分散型電源**: H27以降ほぼ毎年出題（H27前は不定期）→ 最重要テーマに成長
- **電気事業法**: R04から年2回化で出題パターンが分散。関連法規（工事士法・PSE法）からも出題
- **計算問題**: 需要率・負荷率・力率改善は定番。近年は絶縁耐力試験の計算も増加
